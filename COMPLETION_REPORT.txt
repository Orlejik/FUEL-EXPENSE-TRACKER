API ENDPOINTS:
──────────────

Authentication:
  POST   /auth/signup              - Register new user
  POST   /auth/login               - Login user
  GET    /auth/me                  - Get current user

Vehicles (Protected):
  POST   /vehicles/                - Create vehicle
  GET    /vehicles/                - Get all vehicles
  GET    /vehicles/{id}            - Get specific vehicle
  PUT    /vehicles/{id}            - Update vehicle
  DELETE /vehicles/{id}            - Delete vehicle

Expenses (Protected):
  POST   /expenses/                - Create expense
  GET    /expenses/                - Get expenses
  GET    /expenses/?vehicle_id={id} - Filter by vehicle
  GET    /expenses/{id}            - Get specific expense
  PUT    /expenses/{id}            - Update expense
  DELETE /expenses/{id}            - Delete expense
  GET    /expenses/stats/vehicle-stats - Vehicle statistics
  GET    /expenses/stats/total-spent   - Total spent

HOW TO GET STARTED:

Step 1: Start PostgreSQL Database
  Command: docker-compose up -d
  Waits:   ~5-10 seconds for container to be ready

Step 2: Install Backend Dependencies
  Command: pip install -r requirements.txt
  Time:    2-3 minutes

Step 3: Start Backend Server
  Command: python main.py
  Result:  Running on http://localhost:8000

Step 4: Rename Frontend Files
  Command: ren frontend_package.json package.json
  Command: ren frontend_env .env

Step 5: Install Frontend Dependencies
  Command: npm install
  Time:    2-3 minutes

Step 6: Start Frontend Development Server
  Command: npm start
  Result:  Running on http://localhost:3000

Step 7: Access Application
  Frontend:  http://localhost:3000
  Backend:   http://localhost:8000
  API Docs:  http://localhost:8000/docs
