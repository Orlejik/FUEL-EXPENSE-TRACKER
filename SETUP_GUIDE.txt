## 🚀 Quick Start

### Step 1: Start PostgreSQL Database
```bash
cd C:\Users\artiom\Documents\TEstWebSite
docker-compose up -d
```

Wait a moment for PostgreSQL to start completely (check with: `docker ps`)

### Step 2: Install Backend Dependencies
```bash
pip install -r requirements.txt
```

### Step 3: Start Backend Server
```bash
python main.py
```

You'll see:
```
INFO:     Application startup complete
INFO:     Uvicorn running on http://0.0.0.0:8000
```

### Step 4: In a NEW terminal, Setup Frontend
```bash
# Rename files to correct locations
cd C:\Users\artiom\Documents\TEstWebSite
ren frontend_package.json package.json
ren frontend_env .env

# Install React dependencies
npm install

# Start React development server
npm start
```

### Step 5: Access the Application
- Frontend: http://localhost:3000
- Backend API: http://localhost:8000
- API Documentation: http://localhost:8000/docs

## 📚 API Endpoints

### Authentication
- `POST /auth/signup` - Register new user
- `POST /auth/login` - Login user
- `GET /auth/me` - Get current user profile

### Vehicles (require authentication)
- `POST /vehicles/` - Create new vehicle
- `GET /vehicles/` - Get all user vehicles
- `GET /vehicles/{id}` - Get specific vehicle
- `PUT /vehicles/{id}` - Update vehicle
- `DELETE /vehicles/{id}` - Delete vehicle

### Expenses (require authentication)
- `POST /expenses/` - Log new expense
- `GET /expenses/` - Get all expenses
- `GET /expenses/?vehicle_id=1` - Get expenses for specific vehicle
- `GET /expenses/{id}` - Get specific expense
- `PUT /expenses/{id}` - Update expense
- `DELETE /expenses/{id}` - Delete expense
- `GET /expenses/stats/vehicle-stats` - Get stats per vehicle
- `GET /expenses/stats/total-spent` - Get total spent

## 🛠 Troubleshooting

### Database won't start
```bash
# Check if Docker is running
docker ps

# View logs
docker logs fuel_tracker_db

# Stop and restart
docker-compose down
docker-compose up -d
```

### Port already in use
```bash
# Backend on different port
python main.py --port 8001

# Frontend on different port
PORT=3001 npm start
```

### Dependencies issue
```bash
# Clear cache and reinstall
pip cache purge
pip install -r requirements.txt --force-reinstall
```
