⚙️ AVAILABLE ENDPOINTS
======================

Authentication:
  POST   /auth/signup        - Create account
  POST   /auth/login         - Login
  GET    /auth/me            - Get user profile

Vehicles:
  POST   /vehicles/          - Create vehicle
  GET    /vehicles/          - List all vehicles
  GET    /vehicles/{id}      - Get specific vehicle
  PUT    /vehicles/{id}      - Update vehicle
  DELETE /vehicles/{id}      - Delete vehicle

Expenses:
  POST   /expenses/                    - Log expense
  GET    /expenses/                    - List expenses
  GET    /expenses/?vehicle_id=X       - List by vehicle
  GET    /expenses/{id}                - Get expense
  PUT    /expenses/{id}                - Update expense
  DELETE /expenses/{id}                - Delete expense
  GET    /expenses/stats/vehicle-stats - Get stats
  GET    /expenses/stats/total-spent   - Total spent

🚧 TROUBLESHOOTING
==================

Issue: Docker container won't start
Solution: Check Docker is installed and running
  docker version
  docker ps

Issue: "Port already in use"
Solution: Change port or kill process using it
  Backend: python main.py --port 8001
  Frontend: set PORT=3001 && npm start

Issue: npm dependencies fail
Solution: Clean and reinstall
  rmdir /s node_modules
  npm cache clean --force
  npm install

Issue: Database connection error
Solution: Verify PostgreSQL is running
  docker logs fuel_tracker_db
  docker-compose restart

Issue: API returns 401 Unauthorized
Solution: Token expired or missing
  • Clear browser localStorage
  • Login again
  • Check Authorization header
