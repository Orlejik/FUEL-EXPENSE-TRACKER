🚀 QUICK START:
===============

1. Start Database:
   cd C:\Users\artiom\Documents\TEstWebSite
   docker-compose up -d

2. Install & Run Backend:
   pip install -r requirements.txt
   python main.py

3. Setup Frontend (in NEW terminal):
   ren frontend_package.json package.json
   ren frontend_env .env
   npm install
   npm start

4. Access:
   Frontend: http://localhost:3000
   Backend: http://localhost:8000
   API Docs: http://localhost:8000/docs

📊 DATABASE SCHEMA:
===================

Users Table:
  - id (PK)
  - email (unique)
  - password_hash
  - created_at

Vehicles Table:
  - id (PK)
  - user_id (FK)
  - name
  - model
  - license_plate
  - created_at

Expenses Table:
  - id (PK)
  - user_id (FK)
  - vehicle_id (FK)
  - liters
  - cost
  - date
  - created_at
